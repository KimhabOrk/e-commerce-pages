## 🔄 Conversion Details

### What Was Converted

#### React Components → Liquid Templates
```
Next.js App Structure    →    Shopify Liquid
├── /app/page.tsx        →    index.liquid
├── /app/company/about   →    page.about.liquid
├── /app/contact         →    page.contact.liquid
├── /app/services/...    →    page.care-guides.liquid
└── /app/.../...         →    page.responsibility.liquid
```

#### React Features → Liquid Equivalents
```
React Code              →    Liquid Code
├── import/export       →    {% render %} / {% include %}
├── {{ variable }}      →    {{ variable }}
├── {data.map()}        →    {% for item in array %}
├── Tailwind CSS        →    Tailwind CSS (preserved)
├── i18n translations   →    {{ 'key' | t }}
└── onClick handlers    →    HTML forms / links
```
